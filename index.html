PICOSNAKES
Visualisation software for multiple AI methodologies using Snake game.

This project delivered an exploratory piece of software, capable of providing the user with visualisations of the decision-making process of multiple pathfinding AI methodologies. The application allows the user to choose between traditional pathfinding and neural network learning algorithms. The exploratory software was developed in C++ using SFML, Box2D, FANN and Visual Studio 2019 as a cross-platform application.

MIRI (Mid-InfraRed Instrument) Companion App
A digital pamphlet app for the Edinburgh Royal Observatory.

As part of a small agile development team, working on behalf of our clients at the UK-ATC, a mobile application was developed to provide information to guests wishing to learn more about the James Webb Space Telescope. The application is designed to read QR codes placed on displays within the observatory and output enhanced contextual data to the user. The mobile application was developed in C# using Xamarin for Visual Studio 2017 and supported Android 6.0.

PICOBOTS
A single-player, dungeon crawler game.

The purpose of this project was to design, implement and evaluate a fully featured prototype game. This included content such as background music, sound effects, tiled levels, art assets, game engine, physics and AI. Picobots is a mini-machine versus biology dungeon-crawler game from the top-down perspective. The game was developed in C++ using SFML, Box2D, custom game engine and Visual Studio 2017 as a cross-platform application.
